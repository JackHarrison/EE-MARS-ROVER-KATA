## Execution<br>
Run on a webserver with query string of commands<br>

eg.<br>
http://localhost/EE-MARS-ROVER-KATA/index.php?directions=LRBF

## Accepted commands:
* F  = forward
* B = backwards
* L = left
* R = right
*
